devopsarr.sonarr.sonarr_download_client_hadouken module – Manages Sonarr download client Hadouken.

Note

This module is part of the devopsarr.sonarr collection (version 0.6.0).

To install it, use: ansible-galaxy collection install devopsarr.sonarr.

To use it in a playbook, specify: devopsarr.sonarr.sonarr_download_client_hadouken.

Examples

---
# Create a download client
- name: Create a download client
  devopsarr.sonarr.sonarr_download_client_hadouken:
    remove_completed_downloads: false
    remove_failed_downloads: false
    enable: false
    priority: 1
    name: "Hadouken"
    host: "localhost"
    url_base: "/hadouken"
    port: 7070
    use_ssl: false
    category: "tv-sonarr"
    username: "user"
    password: "password"
    tags: [1,2]

# Delete a download client
- name: Delete a download client
  devopsarr.sonarr.sonarr_download_client_hadouken:
    name: Example
    state: absent
