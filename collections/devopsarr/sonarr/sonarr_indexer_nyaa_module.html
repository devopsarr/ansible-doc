devopsarr.sonarr.sonarr_indexer_nyaa module – Manages Sonarr indexer Nyaa.

Note

This module is part of the devopsarr.sonarr collection (version 0.5.0).

To install it, use: ansible-galaxy collection install devopsarr.sonarr.

To use it in a playbook, specify: devopsarr.sonarr.sonarr_indexer_nyaa.

New in devopsarr.sonarr 0.5.0

Notes

Note

  • for authentication, you can set service_account_file using the c(SONARR_URL) env variable.

  • for authentication, you can set service_account_contents using the c(SONARR_API_KEY) env variable.

Examples

---
# Create a indexer
- name: Create a indexer
  devopsarr.sonarr.sonarr_indexer_nyaa:
    name: "Nyaa"
    enable_automatic_search: false
    enable_interactive_search: false
    enable_rss: false
    priority: 10
    api_key: 'test'
    base_url: "https://api.broadcasthe.net/"
    minimum_seeders: 10
    seed_ratio: 0.5
    tags: [1,2]

# Delete a indexer
- name: Delete a indexer
  devopsarr.sonarr.sonarr_indexer_nyaa:
    name: Example
    state: absent
