devopsarr.radarr.radarr_tag module – Manages Radarr tag.

Note

This module is part of the devopsarr.radarr collection (version 1.0.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install devopsarr.radarr.

To use it in a playbook, specify: devopsarr.radarr.radarr_tag.

New in devopsarr.radarr 0.0.1

Synopsis

  • Manages Radarr tag.

Parameters

Parameter

Comments

label

string / required

Actual tag.

radarr_api_key

string / required

API key for Radarr authentication.

radarr_url

string / required

Full Radarr URL with protocol and port (e.g. `https://test.radarr.tv:7878`)

state

string

Create or delete resource.

Choices:

  • "present" ← (default)

  • "absent"

Notes

Note

  • for authentication, you can set service_account_file using the c(RADARR_URL) env variable.

  • for authentication, you can set service_account_contents using the c(RADARR_API_KEY) env variable.

Examples

---
# Create a tag
- name: Create a tag
  devopsarr.radarr.radarr_tag:
    label: default

# Delete a tag
- name: Delete a tag
  devopsarr.radarr.radarr_tag:
    label: wrong
    state: absent
